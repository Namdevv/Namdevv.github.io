# Nam Tran — Portfolio

Personal portfolio of **Nam Tran**, an AI Engineer focused on LLM, vLLM, RAG, and MLOps.

🔗 **Live:** https://namdevv.github.io

## Tech
Pure static site — no build step. `HTML + CSS + vanilla JS`, deployed via GitHub Pages.

```
index.html    structure & content
styles.css    design system, layout, animations
script.js     typing effect, scroll reveal, counters, nav
```

## Local preview
Just open `index.html` in a browser, or serve it:

```bash
python -m http.server 8000
# → http://localhost:8000
```

## Customize
Edit the content directly in `index.html` (projects, links, copy) and the
color tokens at the top of `styles.css` (`:root`).
